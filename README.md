# 灵感之笔

这是一个优雅的诗歌分享平台，致力于传播诗歌艺术之美。

## 功能特点

- 简约优雅的设计风格
- 响应式布局，完美适配各种设备
- 诗歌分类与标签系统
- 用户评论与互动功能
- 优化的阅读体验
- 诗歌收藏功能
- 作者主页展示

## 技术栈

- HTML5
- CSS3
- JavaScript
- FontAwesome（图标库）
